# 🌳 Decision Tree Classifier Project

## 🚀 Overview
This project demonstrates the implementation of a **Decision Tree Classifier**, one of the most fundamental machine learning algorithms used for classification tasks.

The goal is to understand how decision trees work, how they split data based on features, and how they can be used to make predictions.

---

## 🎯 Objectives
- Understand the working of Decision Tree algorithm  
- Train a classification model on dataset  
- Visualize and interpret decision-making process  
- Evaluate model performance  

---

## 🛠️ Tech Stack
- **Language:** Python  
- **Libraries:** pandas, NumPy, scikit-learn, Matplotlib  

---

## 📂 Workflow

### 1. Data Loading
- Imported dataset using pandas  

### 2. Data Preprocessing
- Handled categorical values  
- Prepared features and target variable  

### 3. Model Training
- Used `DecisionTreeClassifier` from scikit-learn  

### 4. Visualization
- Visualized the decision tree structure  

### 5. Evaluation
- Checked model performance using accuracy score  

---

## 📊 Key Concepts Covered
- Decision Tree algorithm  
- Gini Index / Entropy  
- Overfitting and tree depth  
- Feature importance

## 📈 Results
